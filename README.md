# お絵描き大喜利サイト

## メンバー
- s18014 宮里 佑太

## 概要
教科書の落書きの様に、既存の絵や写真に落書きを足していく遊び。
出題者が画像とお題を投稿して、回答者が画像に絵を描き足してお題に答える。

## ターゲット
- 暇な奴
- 落書きが大好きな人

## 機能
- 画像とお題の投稿
- 画像のお絵描き機能
- 回答されたお題の評価機能(Goodボタンなど)
- 人気のお題や回答の検索機能
